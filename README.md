ember-bootstrap-icons
==============================================================================

Use Bootstrap Icons in your Ember project. This addon uses Ember Octane.

```hbs
<BsIcon @name="toggles" />
```


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-bootstrap-icons
```
